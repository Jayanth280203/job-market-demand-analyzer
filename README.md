# Job Market Skill Demand & Salary Intelligence Dashboard

## Project Overview
This project analyzes LinkedIn job postings to identify in-demand skills and high-paying skills using Databricks and Apache Spark.

## Technologies Used
- Databricks
- Apache Spark (PySpark)
- Spark SQL
- Python
- Big Data Analytics

## Dataset
- LinkedIn Job Postings Dataset (Kaggle)
- Tables used:
  - job_skills.csv
  - skills.csv
  - salaries.csv

## Key Features
- Multi-table joins using Spark
- Skill demand analysis
- Salary intelligence by skill
- SQL-based analytics
- Interactive visualizations in Databricks

## Key Insights
- Identified top in-demand skills in the job market
- Analyzed average salary associated with each skill
- Highlighted high-paying skills for career planning

## Use Case
- Career guidance
- Job market analytics
- Big data skill analysis
